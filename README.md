# FeedGet - NLW Return

Widget para feedbacks, desenvolvido durante a NLW Return da Rocketseat

### Funcionalidades

- Coleta de feedbacks
- Captura de tela automática
- Responsivo com acessibilidade

### Tecnologias

- [Vitejs](https://vitejs.dev/)
- [ReactJS](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Express](https://expressjs.com/)
- [Prisma](https://www.prisma.io/)
- [HeadlessUI](https://headlessui.dev/)
- [PhosphorIcons](https://phosphoricons.com/)

### Rodando localmente

Clone o repositório

```bash
  git clone https://github.com/AnderGuedes89/FeedGet.git
```

Acesse a pasta do projeto no terminal

```bash
  cd FeedGet
```

#### Server - NodeJS

```bash
# Acesse a pasta 'server' do projeto
cd server

# Instale as dependências
npm install

# Inicie o servidor
npm run dev

# Acesse http://localhost:3333
```

#### Web - ReactJS

```bash
# Acesse a pasta 'web' do projeto
cd web

# Instale as dependências
npm install

# Inicie a aplicação
npm run dev

# Acesse http://localhost:3000
```
